Prerequisites(Windows)
1) Oracle DataBase (11g XE)
2) Python 3.5.*
3) Django framework
4) Django material python package
5) Microsoft Visual Studio
6) cx_Oracle python extension

Prerequisites(Ubuntu)
1) Oracle DataBase (11g XE)
2) Python 3.5.*
3) Django framework
4) Django material python package
5) cx_Oracle python extension
